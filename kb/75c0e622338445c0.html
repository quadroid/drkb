>> Экспоненциальное распределение
 
Возвращает случайное число, распределенное по экспоненциальному закону распределения

function RandomExponent(ex: Extended): Extended;
{ экспоненциальное распределение }
begin
  Result := - ex * Ln(Random)
end;

Пример использования:

x := RandomExponent(1);
